# C Simple Interest Calculator

A beginner C program that calculates simple interest using user input.

## Formula

Simple Interest = (Principal × Rate × Time) / 100

## Features

- Takes principal amount as input
- Takes rate of interest as input
- Takes time in years as input
- Calculates simple interest
- Displays result with 2 decimal places

## Technologies Used

- C Programming Language
- GCC Compiler

## Example

Enter the principal amount: 1000
Enter the rate of interest: 5
Enter the time in years: 2

The simple interest after 2 years is: 100.00
